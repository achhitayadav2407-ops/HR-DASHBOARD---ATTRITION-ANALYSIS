# HR-DASHBOARD---ATTRITION-ANALYSIS

This  project delivers a comprehensive attrition analysis for a workforce of 1.48K employees, transforming raw data into actionable insights to improve corporate retention strategies. The dashboard tracks key metrics like the overall 16.08% attrition rate, identifying critical risk factors such as high departure volumes in the 26-35 age bracket, the R&D department, and lower salary tiers. By mapping employee demographics, tenure, and role-specific trends, this interactive tool empowers HR teams to make data-driven decisions to mitigate turnover.
